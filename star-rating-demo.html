  • Hover Preview: Hover over stars to see exactly what rating you'll select
  • Visual Feedback: Text shows "Preview: X stars" during hover and "Current: X stars" when not hovering
  • Fractional Ratings: Supports decimal ratings like 3.23 stars
  • Unrated Mode: Shows gray stars when no rating is set
  • Readonly/Disabled: Non-interactive modes for display purposes
  • Customizable: Configurable colors, sizes, and max rating via CSS variables

ðŸŽ¨ CSS Variables

The component uses these CSS variables for customization:

  • --rating-star-color - Color for filled stars (default: #ffd700)
  • --rating-star-unrated-color - Color for unrated stars (default: #d3d3d3)
  • --rating-star-hover-color - Color for hover state (default: #ffed4e)
  • --rating-star-size - Size of stars (default: 24px)
  • --rating-star-spacing - Gap between stars (default: 2px)
